Simple News App to test Gemini Agentic workflow to build the KMP app.  
1. The app was built fully with text prompts using Gemini.
2. Asked Gemini to migrate an old NewsApp (MVVM + Clean Architecture) to KMP to support cross platform
3. It was an iterative process where many issues and crashes came and had to ask Gemini to fix them one by one.
4. Nevertheless it does help in writing a lot of boilerplate code and save lot of dev time with focus lying more on the architecture.
